Tempura-theme-gtk3

This is an attempt at porting lokheed's GTK2 theme Tempura to GTK3.

Download this project as a .zip file Download this project as a tar.gz file

Information

This is a port (and continuation) of the beautiful Tempura theme by lokheed to gtk-3.x.

Installation

To install, create ~/.themes/ in your home directory:

mkdir ~/.themes

Then clone this repository to that location:

git clone https://github.com/hckr/tempura-theme-gtk3 ~/.themes/Tempura

Credits

Tempura Theme originally by lokheed. GTK3 theme based on Numix Frost.
